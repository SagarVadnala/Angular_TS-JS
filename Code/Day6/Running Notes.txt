 Databinding
 - Its a mechanism to bind the data values to html elements & turning user activities to trigger
 - Types of databinding
  Interpolation -  Component   => DOM   {{}}
  Event binding -  Component  <= DOM    ()
  Property binding  - Component => DOM  []
  Two way binding    Component <=> DOM   [(ngModel)]      import FormsModule
  signal 					Compoent <=> DOM    productsize() (input)=productSize.set($event.target.value)
  
  
  Angular Signals offer a more predictable, reactive, and performant way to handle form state compared to the older ngModel–based two-way binding. 
  No zonejs used
  -ngModel relies on dirty checking, zone based change detection, global updates for every input event
  -Signal enable fine grained reactivity, only update the DOM when signal is read, gain performance

---Day6---
Services
- Used to share the common functionalities across the components
- By default services are singleton
> ng g s <ServiceName>

ReactiveX 
- An API for asynchronous programming with observable streams
- Available as Rx.Net, RxJava, RxJs

Observable
- Its a function that return stream of data values to an observer over time 

Observer
- Its a function that consumes the data values sent by observable & it notifies the subscribers
- It has 3 callback methods
  next()  - success callback
  error() - failure callback
  complete()- Indication that no more data is coming

Operator
- It takes 1 observable as input & return another observable after manipulation
  of(), map(), filter(), retry()
  
Steps to implements & call API
- Add a provider in app.config.ts
  provideHttpClient()
- Create a service using a cmd
  >ng g s <serviceName>
- Inject HttpClient class
   httpClient = inject(HttpClient);
- Create function in the service using httpClient that return observable
   getProducts() {
    return this.httpClient.get<Product[]>(`${environment.BASEURL}/api/ProductAPI/getall`)
    .pipe(retry(2));
  }
- Inject the service in the component & call the functions. Also subscribe it
   this.subscriptions.push(this.productService.getProducts().subscribe({
      next: (resp) => {
        console.log(resp);
        this.products = resp;
      }
    }));
