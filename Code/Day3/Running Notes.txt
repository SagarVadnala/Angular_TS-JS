 Databinding
 - Its a mechanism to bind the data values to html elements & turning user activities to trigger
 - Types of databinding
  Interpolation -  Component   => DOM   {{}}
  Event binding -  Component  <= DOM    ()
  Property binding  - Component => DOM  []
  Two way binding    Component <=> DOM   [(ngModel)]      import FormsModule
  signal 					Compoent <=> DOM    productsize() (input)=productSize.set($event.target.value)
  
  
  Angular Signals offer a more predictable, reactive, and performant way to handle form state compared to the older ngModel–based two-way binding. 
  No zonejs used
  -ngModel relies on dirty checking, zone based change detection, global updates for every input event
  -Signal enable fine grained reactivity, only update the DOM when signal is read, gain performance
